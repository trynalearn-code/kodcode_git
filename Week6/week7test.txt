	pass
class Dog(Animal):
	pass


d=Dog()
print(type(d)==Animal #False
print(isinstance(d,Animal) #True
print(issubclass(d,Animal) #True

type checks if the object is literally the same as the class (False). Is instance can look at the hierarchy and see the relationship (will in this case return True).
